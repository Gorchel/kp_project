Содержимое репозитория поместить в корневую папку сервера.
Папка kpdatabase - таблица бд. Ее необходимо разместить в корневой папке для таблиц Mysql.

index.php
Вкладка Contacts открывает окно для добавление записи о пользователе. 
Вкладка Services показывает список существующих записей, упорядоченный по дате. В данном окне организована возможность добавления новой записи, удаления существующей и редактирование.
